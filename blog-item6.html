Express

Express was also built under the principle of API composition. Every piece of software in Express is called middleware, and they are chained in the code in order to process every request.

After starting our node js project and having created our package.json file, install Express in the app directory and save it to the dependency list. For example:

npm install express --save
npm install pug --save

In order to better understand the way we work, we will make a simple example, after creating the index.js file, we copy the following code:

var express = require('express');
var app = express();

app.get('/', function (req, res) {
  res.send('Hello World!');
});

app.listen(3000, function () {
  console.log('Server is up on port 3000');
});

The application starts a server and listens for connections on port 3000. The application responds with "Hello World!" For requests to the root URL (/) or to the root path. For each different access path, it will respond with a 404 Not Found error.
