Create and search lists with Node JS

Steps to create a list of students with their personal data and the cycle in which they are enrolled, then let's review the list by adding the items sought in a second list.

The first thing we will do is create the following lists:

List students

let students = [
  { codigo: '20133',
   name: 'Susan',
   apellido: 'Asade',
   ciclo: '7'
  },
  { ..... },
  { codigo:'20133',
   name: 'Alejandro',
   apellido: 'Fernandez',
   ciclo: '4'
  },
  { codigo:'20135',
   name: 'Sofia',
   apellido: 'Rodriguez',
   ciclo: '10'
  }
  ];

List cycles

let ciclos = [
  { ciclo: '1',
   estudiantes: '0'
  },
  { ciclo: '2',
   estudiantes: '0'
  },
  { ciclo: '3',
   estudiantes: '0'
  },
  { ciclo: '4',
   estudiantes: '0'
  },
  { ..... },
  { ciclo: '10',
   estudiantes: '0'
  }, ];


After declaring the lists we have to go through the list to see the coincidences between the cycles of the students and thus increase the number of students enrolled in each of the cycles.

We have two ways of assigning the values if the second list is not ordered we will have to go through this list inside the other in the following way.

students.forEach(function(item, index){
  for(i=0; i < ciclos.length; i++){
    if(item.ciclo == ciclos[i].ciclo){
      ciclos[i].estudiantes++;
    }
   }
});


In this case as the cycles are sorted according to their position in the list eg cycle 4 is position 3, then cycle n will be the position (n-1).

students.forEach(function(item, index){
  ciclos[item.ciclo-1].estudiantes++;
});
