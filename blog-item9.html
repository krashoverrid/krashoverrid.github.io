Chat with sockets and nodejs

When you want to build an application you have to consider several factors: programming, design, server configuration, among other things.

You can take on all that burden or seek support in existing services and focus on the core: develop your application.

In this post I will show how to create a simple application with nodejs that returns a message and how to leave it hosted in Heroku.

Requirements

  1. Node JS, para decargar.

  2. Heroku CLI, para decargar.

Create una account in Heroku

We enter the Heroku url and then click on "Sing Up"

you register and then add an app:

In this case, since we are not using any database or an extra app, we should not install anything additionally.

Download the following ChatSockets project.

First open the terminal and execute the following:

git clone https://github.com/krashoverrid/Programacion-de-aplicaciones-en-red/tree/master/laboratorio10

we enter the folder that is generated with the following line:

cd laboratorio10

Subiendo todo a Heroku

Having installed Heroku CLI, it is necessary to login with our credentials:

heroku login

now we add the repository the contents of the folder, with the following command:

git add .

we execute the following command:

git commit -am 'socket chat'

Now we can upload the changes to Heroku:

git push heroku master

finally we enter the url generated by Heroku, the application that we have running is the following click to see the demo.
