Post method with Middleware and Node Js

With this application we use the post method and the get method of Middleware with Express in Node js, this will help us to receive data and be able to work the backend and make modifications in the fronted.

We call POST parameters to those that are sent from the client web to the server without making use of the url send, unlike the GET parameters. If you have ever considered how to send POST parameters by Node.js in this article we will teach you how to carry it out. We will also use Express to make development much easier.

The first thing we have to know is that the POST parameters are sent in the same body of the request and not inside the url as the GETs do. When a server receives a POST parameter, it does so in the following way:

application / x-www-form-urlencoded. From here, both the GET and POST parameters have the same format:

var1 = x & var2 = y & var3 = z

When using Express we have a middleware called body-parser, which will help us access the body content of the requests.

If we want to install the body-parser middleware in our project we will have to do it through npm.

$ npm install body-parser

So, before you start, be sure to load the Express middleware and body-parser.

var body_parser = require ('body-parser');
Now we just have to define the Express application 

var app = express ();

Next, we use the middleware body-parser

app.use (body_parser.urlencoded ({extended: true}));
